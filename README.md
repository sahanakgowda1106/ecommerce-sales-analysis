# 📊 E-commerce Sales Analysis & Dashboard

## 🚀 Project Overview

This project presents an end-to-end data analysis of an e-commerce dataset. The objective is to analyze sales performance, identify key trends, and generate actionable business insights using data analytics tools.

---

## 🎯 Objectives

* Analyze sales and profit trends over time
* Identify top-performing products and regions
* Understand the impact of discounts on profitability
* Build an interactive dashboard for business decision-making

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy)
* Matplotlib & Seaborn (Data Visualization)
* Power BI (Dashboard Creation)

---

## 📂 Dataset Description

The dataset contains 1000+ records with the following features:

* Order ID
* Order Date & Ship Date
* Country
* Segment
* Product
* Sales
* Profit
* Discount
* Quantity

---

## 🔧 Data Cleaning & Preprocessing

* Handled missing values and removed duplicates
* Converted date columns to datetime format
* Fixed data types for accurate analysis
* Created new features like Year and Month

---

## 📊 Exploratory Data Analysis (EDA)

Performed detailed analysis to uncover patterns:

* Monthly sales trend analysis
* Country-wise sales and profit comparison
* Product performance analysis
* Segment-wise distribution
* Discount vs Profit relationship

---

## 📈 Key Insights

* 📌 Sales show seasonal trends with peaks in certain months
* 📌 High discounts negatively impact profit margins
* 📌 A few products contribute significantly to total revenue
* 📌 Certain regions outperform others in both sales and profit

---

## 📊 Power BI Dashboard

An interactive dashboard was created to visualize key metrics:

* Total Sales, Profit, and Quantity KPIs
* Monthly sales trends
* Sales by Country and Segment
* Top-performing products
* Discount vs Profit analysis
